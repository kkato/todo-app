# todo-app

This is a todo app developed in Go.

This app has features the following feasures:
- user registration
- login/logout
- create, delete, update todos

I have deployed the app on AWS using EC2 and RDS, so feel free to visit! 

You can access the app from [here](http://ec2-52-195-176-225.ap-northeast-1.compute.amazonaws.com/)
